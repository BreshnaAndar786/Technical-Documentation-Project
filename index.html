Printing to the console

In order to print the console.we use System.out.println.

In know, it is very long and cumbersome, but this is the way it's done. 

            
                public class My main function {
                    /*Java main function example */
                    public static void main(string[] args) {
                      System.out.println("Hello world");
                    }
                }

In this example we are printing out "Hello World" to the console when we run the program.
